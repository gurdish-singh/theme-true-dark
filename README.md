**True Dark** is a sleek and modern theme designed to minimize eye strain, improve focus, and create a visually comfortable experience for users who prefer dark interfaces. Perfect for long hours of work, creative projects, or visionary tasks, True Dark combines elegance with functionality.

## Features

- **Immersive Dark Palette**: Deep colors that highlight content without distraction.
- **Enhanced Focus**: Reduces eye fatigue and allows longer periods of concentration.
- **Professional Design**: Balanced aesthetics suitable for both creative and productivity-focused environments.
- **Motivating and Inspiring**: Encourages a visionary mindset while working.
- **Optimized for Readability**: Clear typography and subtle contrasts for maximum comfort.

## Installation

1. Open **Visual Studio Code**.
2. Go to the **Extensions** panel (Ctrl+Shift+X or Cmd+Shift+X).
3. Search for `"True Dark By Gurdish Singh"`.
4. Click **Install** and activate the theme.

## Usage

- Set **True Dark** as your primary VS Code theme to reduce light exposure.
- Ideal for coding, writing, designing, or any long-term screen tasks.
- Pairs well with productivity extensions and custom editor setups.
